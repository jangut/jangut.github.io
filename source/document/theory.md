2.语法
===

```
1.标题
	一共两种表示，换行等于(=)或减号法；多级井号(-)空格法
2.字体
	双星**加粗**,单星*倾斜*，双波~~删除线~~,双等==高亮==
	<u>下划线</u>，`code`
```

字体演示:**加粗**,*倾斜*，~~删除线~~,==高亮==，<u>下划线</u>，`code`

```
3.段落
	在标题和内容之间留空行；在列表前后留空行；在代码块前后留空行
4.分割线
	三星号，三减号
5.脚注
	[^1]
```
这是一个脚注[^1]

```
6.列表（tab嵌套，>段落）
	无序减加空格，有序数点加空格，选框减空左方空右方空（未完x）
```

- 无序列表
	1. 嵌套
	2. 数字
		>这里可以放一个段落
	3. 列表
		- [ ] 框 
		- [x] 未完成

```
7.引用块
	由>开头
8.代码框
	三点加语言
9.插入链接
	[].()或<>方点圆或者尖;双方链接文件；#链接标题
```
